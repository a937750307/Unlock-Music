# Unlock-Music

Unlock Music是一款音乐解锁程序,它可以帮助用户转换QQ音乐、Moo音乐、网易云音乐、虾米音乐、酷狗音乐和酷我音乐等等平台中的付费歌曲,让用户可以在其它的软件或者设备中播放这些音频。

# 演示地址

https://unlock-music.937788.xyz/

# 功能特性

支持以下格式解锁：

QQ 音乐 (.qmc0/.qmc2/.qmc3/.qmcflac/.qmcogg/.tkm)

Moo 音乐格式 (.bkcmp3/.bkcflac/…)

QQ 音乐 Tm 格式 (.tm0/.tm2/.tm3/.tm6)

QQ 音乐新格式 (.mflac/.mgg/.mflac0/.mgg1/.mggl)

QQ 音乐海外版JOOX Music (.ofl_en)

网易云音乐格式 (.ncm)

虾米音乐格式 (.xm)

酷我音乐格式 (.kwm)

酷狗音乐格式 (.kgm/.vpr)

Android版喜马拉雅文件格式 (.x2m/.x3m)

咪咕音乐格式 (.mg3d)
